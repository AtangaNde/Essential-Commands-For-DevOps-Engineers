1. gh workflow list – List all GitHub Actions
workflows

2. gh workflow run my-workflow.yml – Run a specific
workflow

3. gh run list – List all workflow runs

4. gh run view <run_id> – View details of a workflow
run

5. gh run cancel <run_id> – Cancel a running
workflow

6. gh run rerun <run_id> – Rerun a failed workflow

7. gh secret list – List GitHub repository secrets

8. gh secret set MY_SECRET --body "value" – Set a
secret in GitHub Actions

9. gh variable list – List environment variables in
GitHub Actions

10. gh variable set MY_VAR --body "value" – Set an
environment variable

13. gitlab-runner stop – Stop GitLab Runner

14. gitlab-runner verify – Verify if runners are

Configuration

Run pipeline locally

20. circleci project list – List all projects in
CircleCI

21. circleci setup – Configure CircleCI for the first
time

22. circleci build – Run a build locally
    
23. circleci config process .circleci/config.yml –
